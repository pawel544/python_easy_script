# 📊 Analiza Sprzedaży — Python + Pandas

Ten skrypt analizuje dane sprzedaży produktów z pliku `sales_data.csv`, generuje raport tekstowy oraz tworzy wykres słupkowy średnich przychodów miesięcznych.

## 🔧 Wymagania

- Python 3.7+
- Biblioteki:
  - pandas
  - matplotlib
