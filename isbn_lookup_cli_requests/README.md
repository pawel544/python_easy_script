# 📚 ISBN Lookup CLI

Prosty terminalowy program w Pythonie, który wyszukuje dane o książce na podstawie numeru ISBN (13 cyfr), korzystając z Google Books API.

## 🔧 Wymagania

- Python 3.x
- Biblioteka `requests`
- Dostęp do Internetu

## ▶️ Jak używać

```bash
python isbn_lookup_cli.py
